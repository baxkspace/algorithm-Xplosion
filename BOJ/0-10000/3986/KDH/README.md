## BOJ 3986 좋은 단어 KDH

## 문제 요약
- N개의 단어(문자열)이 주어진다.
- 단어 속 'A', 'B'가 모두 올바르게 짝지어지면 그건 좋은 단어이다.
- N개의 단어 중 좋은 단어의 개수를 리턴한다.

## 풀이 방법
- 'A', 'B'는 레벨이 다른 괄호쌍이라고 생각하면 된다. `({()})`과 같은 괄호쌍은 올바른 괄호쌍이고, `({)}`과 같은 괄호쌍은 올바르지 않은 괄호쌍이다. 

- 올바른 괄호쌍과 유사한 풀이법으로 스택을 사용한다.
- 다음으로 들어올 문자가 이미 st.top()에 존재하면, 같은 레벨의 괄호쌍끼리 모였으므로(같은 문자끼리 모였으므로), st.pop()을 해 준다.
- 다음으로 들어올 문자가 st.top()의 값과 다르다면, 계속 push해 준다.
- 문자열의 끝까지 위 과정을 반복했을 때, 스택에 문자가 남아있지 않으면 좋은 단어가 된다.
