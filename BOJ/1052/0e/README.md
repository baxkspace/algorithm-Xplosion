## How to solve
bin(N) : 1의 개수만큼 물병이생김
만약 19라고 했을때 10011(16+2+1) 3개가 생김

1의 개수가 K보다 작거나 같을때까지 while문 돌아감
for문으로 bin(N)의 1의 자리수부터 검사
1이나오면 answer와 N에 자릿수만큼 2의제곱해서 더해줌
break
다시 1의 개수와 K값 비교
