## LeetCode 20.Valid Parenthese KDH

## 문제 요약
- 괄호 종류 `()`, `[]`, `{}`
- 주어진 문자열이 올바른 괄호쌍이라면(if it is valid) true, 아니라면 false 출력

## 풀이 방법
- 스택 하나에 주어진 문자열의 문자(괄호)를 push하거나 pop하면서 유효성을 체크할 거임
- 열린 괄호 `(`, `{`, `[`가 주어지면 스택에 push, 닫힌 괄호 `)`, `}`, `]`가 주어지면 top 원소와 비교해서 같은 종류의 괄호라면 pop 한다.
- top 원소와 비교했을 때 다른 종류의 괄호이거나, 문자열 끝까지 순회했을 때 스택이 비어있지 않다면 invalid -> false를 출력한다.
- 문자열을 끝까지 순회했을 때 stack이 비어있고, 순회하는 도중에 invalid 연산이 발생하지 않았다면 true를 출력한다.
